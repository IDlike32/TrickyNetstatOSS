# Tricky NETSTAT OSS

This is a tool to patch the netstat in your local cmd display.

这是一个工具用于伪造netstat的命令行输出结果。

It will filter and frogen the output of the netstat command.

它会过滤例和伪造netstat的命令输出。

### What it do? -它做什么

Filet netstat.exe output port like 135/139

过滤像135，139这样的端口

### How to use it? -怎么用

1. Download the latest release from the release page. 
2. run TrickyNetstatOSS.exe
3. Agree administrator permissions
4. Enjoy!

1. 下载
2. 运行
3. 授予管理员权限
4. 享受吧！

### How to uninstall it -怎么卸载

use netstat -u, then you should del C:/Windows/netpath.exe

使用 netstat -u ，然后自己删除C:/Windows/netpath.exe
